# Django Blog Project

A **full-featured Django Blog Application** built with Django, featuring a complete blogging platform with authentication, role-based authorization, an administrative dashboard, blog and category management, commenting system, search functionality, media handling, and production-ready deployment configuration.

---

## ✨ Features

### 🌐 Public Website

- Responsive homepage
- Featured blog posts section
- Recent articles section
- Blog categories
- Browse posts by category
- Individual blog post pages
- Search functionality
- Social media integration
- Custom 404 error page
- Template inheritance
- Static and media file support

---

### 🔐 Authentication & Authorization

- User registration
- User login
- User logout
- Secure authentication system
- Authorization with protected routes
- Login-required views
- Django Groups & Permissions
- Role-based access control

---

### 📊 Dashboard

- Manager Dashboard
- Editor Dashboard
- Dashboard statistics
- Category management
- Blog post management
- User management *(Managers only)*
- Sidebar navigation
- Active navigation highlighting

---

### 📝 Blog Management

- Create blog posts
- Edit blog posts
- Delete blog posts
- Automatic unique slug generation
- Automatic author assignment
- Featured posts support
- Category relationships

---

### 📂 Category Management

- Create categories
- Edit categories
- Delete categories

---

### 💬 Comment System

- Authenticated users can leave comments
- Display comments under blog posts
- Integrated comment management

---

### 🚀 Deployment

- Production-ready configuration
- Deployment checklist
- Ready for hosting on production servers
